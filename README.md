# ninjasoapopera
Website for the music band NinjaSoapOpera

# Tech stack
🏗️ The Tech Stack
1. Frontend (Client-Side)
Language: Native JavaScript (Vanilla JS), HTML5, CSS3.
Hosting: Vercel

2. Backend (Server-Side API)
Framework: C# (.NET 8 Web API)
Containerization: Docker
Hosting: Render.com (Free Web Service Tier)

3. Database & Storage
Audio Storage: Google Drive API (15GB Free)
Database: Google Sheets API
